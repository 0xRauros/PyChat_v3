# PyChat_v3
Python 3 P2P chat application, secured with AES encryption

## [Version 3.0.1]
Only a concept application. Core code is missing, only the front-end works.


Installation:

Linux: 
```Shell
pip3 install -r requirements.txt
python3 main.py
```

Windows:
```Shell
pip3 install -r requirements.txt
py main.py
```
